# DFWIN Corporate Website & Sourcing Platform

A premium, high-performance static website and interactive New Product Introduction (NPI) sourcing portal designed for DFWIN International's B2B supply chain operations. 

## Features
- **Premium B2B Aesthetic**: Dark mesh theme optimized for enterprise clients, featuring subtle micro-animations and clean layouts.
- **Interactive Pathfinder Cards**: Centered on the homepage, routing clients directly to NPI Engineering, the Product Catalog, or the Advisory Consultation portal.
- **Dynamic Product Catalog**: Client-side interactive catalog categorizing standard Marine, Technology, and Hospitality products with seamless filtering.
- **Solution Portal**: A multi-step Alpine.js powered interactive wizard that guides prospective clients through detailing their sourcing objectives and product specifications.
- **Responsive Navigation**: Persistent, glassmorphic header navigation unified across all pages for a cohesive user journey.
- **B2B Case Studies**: Dedicated sub-pages highlighting operational success stories with key industry partners (e.g. fleet-wide refits and specialized luggage cage contracts).

## Directory Structure
- `/index.html` - Homepage
- `/catalog.html` - Interactive product catalog
- `/about.html` - Executive history and team summary
- `/network.html` - Supply infrastructure details
- `/solution-portal.html` - Quote and NPI engineering request portal
- `/thank-you.html` - Confirmation page for submissions
- `/case-studies/` - Standalone B2B operational case study sub-pages
- `/images/` - High-resolution corporate logo and placeholder image assets
